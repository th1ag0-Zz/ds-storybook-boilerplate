# Como rodar o projeto localmente?

## Pré-requisitos

- Node.js (versão 18 ou superior)

## Instalação das dependências

Para instalar todas as dependências, execute no diretório raiz:

```bash
npm install
```

## Rodando o storybook

Para rodar o storybook, use o comando:

```bash
npm run storybook
```

## Scripts disponíveis

Esses scripts foram criados para automatizar tarefas comuns de desenvolvimento, como build, testes, formatação e atualização de tipos, garantindo padronização e agilidade no fluxo de trabalho.

### Storybook (Desenvolvimento)

Scripts relacionados ao desenvolvimento:

`npm run storybook` Inicia o Storybook localmente (porta 6006).

`npm run build-storybook` Gera a build estática para deploy.

### Geração de Templates

Automação para criação de componentes:

`npm run generate-component` Cria um novo componente utilizando o gerador Plop, baseado no template definido em ./generators/plopfile.js.

### Build e Tipos

Scripts responsáveis pela build geral do projeto, Tailwind e geração de tipos:

`npm run build` Executa todo o processo de build: gera arquivos globais de exportação e compila o bundle via Rollup.

`npm run generate-exports-files` Gera os arquivos globais de exportação do projeto.

`npm run build:types` Gera os arquivos .d.ts com base no tsconfig.build.json.

### Qualidade de Código

Scripts de lint e formatação:

`npm run lint` Roda o linter (Biome).

`npm run lint:fix` Corrige problemas automaticamente.

`npm run format` Formata o código usando Biome.

### Outros

Comandos auxiliares:

`npm run prepare` Instala os hooks do Husky (executado automaticamente durante npm install).
