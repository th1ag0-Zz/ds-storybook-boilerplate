# Monorepo DS Boilerplate

<img src={bannerImage} alt="Banner Placeholder" />

O sistema de design para construir interfaces modernas e consistentes.

Os componentes, estilos e utilitários do Monorepo DS Boilerplate são o núcleo do design system compartilhado em todas as aplicações.

Este storybook representa os componentes encontrados no repositório `@monorepo-ds-boilerplate/react`.

Sua organização está alinhada com os princípios de Design Systems modernos, utilizando:

- Compound Components
- Tokens de Design
- Componentes Reutilizáveis
- Documentação Detalhada
